글자모양 태그

우리가 지금 배우고 있는 HTML은 웹 스크립트

mark라는 태그는 마치 노란색 형광펜으로 줄친 효과.

고려시대 다음은 朝鮮時代조선시대

            
            public class HelloWorld{
                public static void main(String [] args){
                    System.out.println("Hello, World");
                }
            }

웹 브라우저에서 다시 불러오기를 하려면 F5키를 누르면 됩니다.

테스형은 소크라테스가 말하기를 너 자신을 알라고 했습니다.
